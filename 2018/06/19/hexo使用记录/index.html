hexo安装

必要条件

需要先安装这两货：

  • Node.js
  • Git

如果已经安装好了，仅仅需要通过npm安装hexo：
npm install -g hexo-cli

初始化

首次安装hexo，运行下面的命令来初始化hexo到目标文件夹.

hexo命令

创建文章\页面\草稿:
hexo new [layout] <title>

将文章从_draft移至_posts：
hexo publish [layout] <title>

生成静态页面：
hexo generate

本地部署,端口4000 预览效果：
hexo server

清除无用标签，分类等:
hexo clean

部署到github：
hexo deploy

主题

  1. 下载主题，放在theme目录下。
  2. _config.yml中修改主题名称

例子：使用NexT主题

优化

评论系统

使用Disqus配置评论功能，由于next主题原生支持，直接修改配置：

  1. 注册Disqus
  2. 修改_config.yml配置 disqus_shortname: your-disqus-shortname
  3. 评论需要科学上网，为方便国内访客，准备搭VPS封装Disqus Api做代理，计划中。

数据统计与分析:

使用腾讯分析，登陆腾讯分析获取ID，
由于next已集成，直接修改配置：
tencent_analytics: <ID>

搜索服务

使用Algolia:

  1. 注册Algolia
  2. 安装Hexo Algolia npm install --save hexo-algolia
  3. 获取key，更新站点配置

  4. 更新index

  5. 打开algolia开关

分享服务

使用百度分享，next已集成，仅仅需要做如下配置：

# hexo # blog
